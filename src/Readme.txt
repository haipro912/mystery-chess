Installation instruction:
1. Supported Operating System:
   Any Operating System that has java 1.5 or higher installed
2. Installation steps:
    a. Copy the jar file (e.g. MysteryChess-1.9.2.jar) 
       and the script files (Run.bat and Run.ksh) into your hard disk
    b. Execute the application:
        + On Windows: Execute this file Run.bat
        + On Linux or Unix: Execute this file Run.sh
    c. If you failed to start up the application, edit the Run.bat (or Run.sh) 
       file to correct the path of java and repeat step c.
